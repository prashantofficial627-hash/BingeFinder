# 🍿 BingeFinder

## 📌 Project Description

BingeFinder is a web application that allows users to search for movies and web series in a fast and interactive way. Users can explore different titles and view important details such as name, release year, type, and poster.

The application is designed to help users quickly find content to watch based on their interests, making the browsing experience simple and enjoyable.

---

## 🎯 Objective

The objective of this project is to:

* Fetch movie and web series data from a public API using JavaScript
* Allow users to search and filter content
* Implement sorting for better user experience
* Display results in a clean and visually appealing interface

---

## 🔗 API Used

OMDB API

👉 https://www.omdbapi.com/

## 🚀 Features (Planned)

### 🔍 Search Functionality

### 🎬 Display Results

* Show movie/series title

### 🎭 Filtering

* Filter results by type (Movie / Series)
